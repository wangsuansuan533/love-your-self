# 爱你老己 - 心理锚点网页

职场人士的"心理锚点"网页，一个不占用大脑带宽的、可呼吸的桌面伴侣。

## 🌟 功能特点

- **初见星球**：选择3件物品作为你的心理锚点
- **每日惊喜**：随机呈现滑动条、二选一或五感盲盒交互
- **AI生成指南**：基于Gemini API的深层关联治愈建议
- **治愈空间**：自适应的动态4K背景视频
- **莫兰迪色系**：柔和的视觉体验
